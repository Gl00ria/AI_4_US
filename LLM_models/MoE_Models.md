# What's a Mixture Of Experts Model 'MoE'?

An LLM Architecture that utilizes multiple specialized submodels, known as experts, to efficiently process tasks.

## Pros

- _Efficiency_: A processing time since it reduces the number of parameters activated during the inference.
- _Scalability:_ Allows you to run Large Models without the dramatic increase in the computational cost.
