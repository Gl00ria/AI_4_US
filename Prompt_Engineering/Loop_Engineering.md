### What's Loop Engineering?

In normal Prompt Engineering you send a prompt to the LLM and Tweak it if the output from the LLM is not what you expect. You keep doing the same thing until you reach the goal.
BUT... What if you replace yourself with an **AI Agent**? That's it...
A Loop Engineering is, handing an AI Agent the task to loop instead of you getting dizzy:

1. Sends the Task to the LLM
2. Checks the output
3. Compare the output with the goal
4. Loop through the above until the goal is met

# A perfect Loop has the below

1. A stop condition: so the loop always ends
2. Context Management: so the agent doesn't drift
3. Error Recovery: Mistakes become a feedback instead of deadends
