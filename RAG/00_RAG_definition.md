#### What is Retrieval Augmented Generation (RAG)?

An Architecture used to help **LLMs** to provide better responses by using realative information from data sources and reducing the chances that an **LLM** will sensitive data or hallucinate or misleading information.

**OK IN SHORT:**
Enhancing the model's response by integrating an external **database** for the model to query in order to pull additional context or information.

#### Why RAG?

To provide richer context for model response, useful for providing detailed knowledge.
