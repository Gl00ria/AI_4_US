# Interpreted-Context-Methdology VS Open Knowledge Format

## [ICM](https://arxiv.org/pdf/2603.16021) by Clief

### Purpose: 'Controls AI Workflow'

Organize AI workspaces and multi-step agent workflows using folders and Markdown instead of orchestration frameworks.

- Breaks work into stages
- Keep context local to each stage
- Store Info as a Markdown files
- Use the File System as a memory
- Avoid complicated multi-agent workflow when a sequential workflow is enough

### Helpful for

- Context Engineering
- Workflow Design

## [OKF](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md) by Google

### Purpose: 'Controls AI Knowledge

A structured specification format for describing knowledge, requirements, schemes, and machine-readable information. So, Instead of organizing work, it organizes knowledge.

### Helpful for

- Specifications
