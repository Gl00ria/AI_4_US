### What is Claude Cowork?
Think of it as an Automation Script :). It does the work that you manually have to do it yourself.

### What is Cowork's Skill?
Think of the Skill as a reusable playbook (folder of files & resources) that teaches Claude how to do a specific task. So Whenever you start a task that matches that skill, Claude will load that playbook and starts to follow it.

### What Cowork's Skill contain?
- **Instructions (SKILL.md) file:** Describes what this skill does.
- **Assents:** The material the skill uses to produce the final output.
- **References:** Examples of the desired output, style guidelines. Basically, whatever that teaches Claude "What a Good" looks like.
- **Scripts:** For Claude can run for the parts of the process the same way every time.


### What Cowork's Plugins?
Plugins are a set of SKILLS designed around a specific job.


>  A Cowork project's memory stays inside that project, while chat memory applies across all your conversations
